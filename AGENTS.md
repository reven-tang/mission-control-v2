## Gate Checks

| 阶段 | 通过标准 | 阻断处理 |
|------|---------|---------|
| SPEC | proposal + design + tasks 存在 | 补齐后重试 |
| BUILD | `npm test` pass + coverage ≥ 80% | 修复后重试 |
| VERIFY | P0 fixed | 修复后重试 |
| QA | 0 console error | 修复后重试 |
| **VIDEO** | **MP4 存在 + 时长 ≤ 60s** | **重新渲染** |
| **PUBLISH** | **人工确认** | **等待用户** |

## 小红书自动化调研（Phase 2 任务 3）

### 调研结果

| 方案 | 状态 | 备注 |
|------|------|------|
| 官方 API | ❌ 无公开视频发布 API | — |
| myaibot.vip | ⚠️ 第三方 API | 需评估稳定性 |
| 易媒助手 | ⚠️ 第三方工具 | 有封号风险 |
| 自建 RPA | ❌ 不推荐 | 高风险 |

### 当前推荐方案

**人工发布** — 视频生成后：
1. 通知用户（飞书/微信）
2. 用户扫码登录创作者中心
3. 手动上传视频
4. 填写标题、标签、描述

### 风险提示

- 第三方自动化工具可能违反小红书规则，有封号风险
- 批量发布建议 ≤5 条/天
- 新账号建议 1-2 条/天
